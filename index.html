a CKEditor plugin to easily add code into your articles

This is a CKEditor plugin that provides an IDE-like features and creates the good syntax to load your syntax highlighter. The IDE-like features are possible owing to the ACE Editor plugin.

It came from my personal need. I am a (little) blogger and I want to write code articles and tutorials. However I did not found any decent plugins to add code blocks into posts so I decided to create mine.

pbckcode is intended for writer, bloggers, coders, developers, everbody ! so feel free to try it !
